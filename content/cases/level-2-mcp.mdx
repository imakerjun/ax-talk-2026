# MCP stdio로 AI 자산 헤쳐모여

> 참고: [흩어져 있는 AI 자산, 'MCP stdio'로 헤쳐모여!](https://techblog.woowahan.com/25986/) (조현석, 2026.03)

---

## 문제

- 프롬프트, 스킬, 규칙이 개인 메모장에 흩어져 있음
- 잘 쓰는 사람의 노하우가 공유는 되지만, 다른 사람이 그대로 쓸 수 없음
- IDE, 도구가 바뀌면 자산도 사라짐

---

## 해결: MCP stdio 기반 중앙화 + npm 배포

- 팀의 AI 자산(프롬프트, 스킬, 규칙)을 하나의 패키지로 관리
- `npm install` → Cursor, Claude Code 등 어떤 IDE에서든 자동으로 최신 자산 적용
- 버전 관리로 자산의 변경 이력 추적

---

## 포인트

"잘 쓰는 사람의 노하우"가 패키지가 되면, 모든 팀원이 같은 수준으로 시작.

공유가 아니라 **설치**로 전파.

**노하우가 패키지가 되면 — 그게 축적이다.**
