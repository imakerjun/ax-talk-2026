# AX, 반복에서 축적으로

> 우아한형제들 사례 중심 | 2026.03.25 삼성SDS 특강

AI 활용의 진짜 전환점은 반복을 축적으로 바꾸는 구조를 만드는 것입니다.

## 발표 내용

| 섹션 | 키워드 |
|------|--------|
| 인트로 | 바이브 코딩 vs 에이전틱 엔지니어링, "공유 ≠ 변화" |
| 개인 → 도구 | AI 테스트 자동화 플러그인 |
| 팀 → 자산 | MCP 자산 중앙화, 디자인팀 스컬피 원형, 기록→검증→도구 파이프라인 |
| 조직 → 문화 | AI Dive Deep — 비개발 직군까지 확산 |
| 마무리 | 오늘 돌아가서 할 수 있는 것 |

## 로컬 실행

```bash
npm install
npm run dev
```
